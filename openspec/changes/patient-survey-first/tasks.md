# Tarefas para 'patient-survey-first'

## Tarefas de Desenvolvimento

1.  **Configurar a rota e a interface inicial do questionário no `survey-patient`.**
2.  **Desenvolver o componente do questionário de 7 perguntas.**
    *   Implementar a navegação entre as perguntas.
    *   Integrar os controles de entrada para as respostas.
    *   Garantir a acessibilidade (WCAG 2.1 Level AA) dos componentes.
3.  **Implementar a lógica para coletar e armazenar as respostas do paciente temporariamente.**
4.  **Desenvolver a tela de resumo pós-questionário.**
    *   Exibir um resumo simplificado com base nas respostas.
5.  **Integrar com o serviço de backend para envio das respostas (se necessário para o resumo inicial).**
6.  **Escrever testes unitários e de integração para a nova funcionalidade.**

## Tarefas de Validação

1.  **Validar a conformidade com WCAG 2.1 Level AA.**
2.  **Testar a funcionalidade de ponta a ponta no `survey-patient`.**
