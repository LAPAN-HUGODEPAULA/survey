# Survey Builder Navigation Guide

## Overview

The Survey Builder has been updated with a new navigation system that provides better administrative workflows and easier access to all features.

## What's New

### 1. Task-Based Dashboard
The new dashboard focuses on tasks rather than data summaries. You'll see large, clear buttons for your most common activities:

- **Create Survey**: Start a new questionnaire from scratch
- **Edit Existing Survey**: Work on questionnaires you've already created
- **Quick Prompts**: Access AI prompts for enhanced responses
- **Manage Personas**: Configure specialized AI personas

### 2. Persistent Navigation
Navigation remains visible across all sections, so you never lose context:

- **Desktop**: Sidebar on the left shows all sections
- **Mobile**: Bottom navigation bar for easy thumb access
- **Tablet**: Adaptive sidebar that adjusts to screen size

### 3. Breadcrumb Navigation
Easily see where you are and navigate back:

```
Dashboard → Questionários → Editar questionário
```

Click any breadcrumb to jump directly to that section.

## Navigation Flows

### Creating a Survey with Prompts

1. Start at the **Dashboard**
2. Click **Create Survey**
3. While editing, click **Quick Prompts** if you need AI assistance
4. Create your prompt and automatically return to your survey
5. Continue building your questionnaire

### Managing Personas

1. From any screen, access **Personas** in the navigation
2. Create new personas or edit existing ones
3. Personas automatically appear in your survey options

### Using Recent Activity

The dashboard shows your recent activity at the bottom:
- Recent survey updates
- New prompts created
- Shared personas

Click any item to jump directly to that content.

## Quick Reference

### Keyboard Shortcuts
- `Ctrl + B`: Toggle navigation (desktop)
- `Escape`: Go back to previous section
- `Ctrl + K`: Quick search (coming soon)

### Mobile Gestures
- Swipe left/right: Navigate between sections
- Long press: Context menu for items
- Pull down: Refresh content

### Common Tasks

| Task | Desktop Path | Mobile Path |
|------|-------------|-------------|
| Create Survey | Dashboard → Create Survey | Dashboard → (+) Create |
| Edit Survey | Dashboard → Edit Survey | Dashboard → Questionários → Edit |
| Add Prompt | Survey Editor → Quick Prompts | Survey Editor → Quick Prompts |
| Manage Personas | Dashboard → Personas | Bottom nav → Personas |
| Export Data | Dashboard → Import/Export | Dashboard → Import/Export |

## Tips and Best Practices

### For Power Users
1. Use keyboard shortcuts for faster navigation
2. Customize your dashboard layout (coming soon)
3. Use keyboard shortcuts in forms (Tab, Ctrl+S, etc.)

### For New Users
1. Start with the **Create Survey** button
2. Follow the on-screen guidance in forms
3. Use the breadcrumb trail to understand where you are

### For Mobile Users
1. The bottom navigation is always accessible with your thumb
2. Swipe gestures will make navigation even faster
3. The interface adapts to your screen size automatically

## Troubleshooting

### Can't find a section?
- Check the sidebar or bottom navigation
- Use the breadcrumbs to navigate back to the dashboard
- Contact support if you still can't find what you need

### Navigation not working?
- Try refreshing the page
- Clear your browser cache
- Check if you're using an unsupported browser

### Form changes lost?
- The builder auto-saves your work locally
- Look for the "Changes not saved" message
- Click "Save" to publish changes
